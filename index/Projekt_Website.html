Wie ist es zu dieser Website gekommen?


Die Idee zu einer Website war schon lange vorhanden, mich reizte es mich mit dem Webdevelopment auseinanderzusetzen und zu erlernen, was es dazu alles braucht und welche Probleme auf dem Weg zur eigenen Website auftauchen.

Alles fing mit einer einfachen HTML Website an, diese bestand aus einem HTML-Dokument und wurde auch nur durch HTML designt, also ohne CSS. Schnell wurde mir aber klar, dass eine Website nur mit HTML nicht so aussieht, wie ich es mir vorgestellt hatte.

Das hat mich dazu bewegt mich weiter mir CSS zu befassen, dabei ist HTML im grunde eine Programmiersprache für die Infhalte auf der Website, CSS wird in den meisten Fällen für die Gestaltung verwendet.

Nachdem ich mithilfe einiger Tutorials in der Lage war eine Funktionierende Website in VS-Code zu schreiben, ein Texteditor von Microsoft, war für mich schnell klar, das diese Seite auch erreichbar sein sollte.

Somit habe ich mich noch weiter mit dem eigentlichen Hosting einer Website auseinandergesetzt, mit allem was dazu gehört
